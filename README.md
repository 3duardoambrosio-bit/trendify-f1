# Trendify Fase 1

Sistema de automatización de e-commerce para Fase 1.

## Estructura

- `buyer/`: Evaluación determinística de productos (Buyer Block)
- `infra/`: Infraestructura compartida (config, logging, métricas, health)
- `config/`: Configuración en YAML + variables de entorno
- `docs/`: Especificaciones y documentación
- `ops/`: Checklists y runbooks operativos
- `scripts/`: CLIs y scripts auxiliares
- `shared/`: Código común reutilizable
